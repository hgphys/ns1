# ネットワーク科学と経済Ⅰ

金沢学院大学 経済情報学部「ネットワーク科学と経済Ⅰ」（担当: 後藤弘光）の演習教材リポジトリです。

対象: 経済情報学部 4 年次

> 続編: [ネットワーク科学と経済Ⅱ](https://github.com/hgphys/ns2)

---

## 授業概要

グラフ理論の基礎からネットワーク分析の実践までを Python（[NetworkX](https://networkx.org/)）を使って学ぶ。「つながり」のデータサイエンスとしてのネットワーク科学の視点から、世間の広さと狭さ、格差社会、グローバル経済といった社会課題を考察する。

## 到達目標

- ネットワークデータの表現法と扱い方を説明できる
- 代表的なネットワーク特徴量（次数・経路長・クラスタ係数・中心性・連結性）を用いて、ネットワーク分析ができる
- ネットワーク科学の視点から格差社会やグローバル経済について考察できる

## 教材構成

各回は以下の3点セットで構成されます。

1. **要点まとめスライド（PDF）** — 授業中の主たる説明資料。Google Drive で受講生に共有
2. **演習ノートブック (`ns1a_XX.ipynb` / `ns1b_XX.ipynb`)** — 問題文と、`%run` で実行する `.py` スクリプトの提示
3. **問題用 `.py` スクリプト** — 学生はノートブックの指示に従ってスクリプトを編集し、実行結果から問題の答えを導く

実践演習回（第5〜7回）では、授業中に Google フォームで収集した受講生間の友人関係データを題材として用います。

## Git のインストール

本教材は GitHub で配布しているため、ダウンロードには Git というツールが必要です。

- Windows: [git-scm.com/download/win](https://git-scm.com/download/win) からインストーラーを取得し、オプションはすべて既定のまま進めてください。インストール後、「Git Bash」を起動して `git --version` が表示されれば完了です。
- macOS: ターミナル.app で `xcode-select --install` を実行してください。ダイアログが出たら「インストール」を押します。すでに入っている場合は `already installed` と表示されるので、そのまま進んで構いません。`git --version` でバージョンが出れば完了です。

## 環境構築（初回のみ）

### 1. Anaconda のインストール

以下のサイトを参考にインストールしてください。

- [Python 環境構築ガイド](https://www.python.jp/install/install.html)
- [Anaconda](https://www.anaconda.com/products/distribution)

> ユーザ名が日本語の場合、インストール先のパスに注意してください。

### 2. 仮想環境の作成

```bash
conda create -n ns_env python=3.11 -y
```

### 3. 仮想環境の有効化

```bash
conda activate ns_env
```

プロンプトに `(ns_env)` と表示されれば OK です。

### 4. 必要パッケージのインストール

```bash
pip install -r requirements.txt
```

> この環境は「ネットワーク科学と経済Ⅱ」でも共通で使用します。

## 課題の取り組み方

各回のフォルダには以下のファイルが含まれます。

```
ns1a_XX/
├── ns1a_XX.ipynb       # 課題本体（Jupyter Notebook）
├── ns1a_XX_01.py       # 問題1用スクリプト
├── ns1a_XX_02.py       # 問題2用スクリプト
├── ns1a_XX_03.py       # 問題3用スクリプト
├── img/                # 画像ファイル（ある場合）
├── edges/              # 辺データファイル（ある場合）
└── data/, data0X/      # データ配置用フォルダ（ある場合）
```

> `data/`, `edges/`, `data0X/` フォルダは演習で利用するデータを配置するためのものです。空フォルダを保持するために `.gitkeep` という空ファイルが入っていますが、無視して構いません。

### 実践演習データの配布について（ns1a_05〜ns1a_07）

実践演習では、受講生間の友人関係データ（Google フォームで授業中に収集）を利用します。

- **ns1a_05** — Moodle コースから配布される `data_merged.csv` を `ns1a_05/data/` フォルダに保存してから取り組んでください。
- **ns1a_06, ns1a_07** — `ns1a_05` で配布されたデータから派生するデータを利用します。Moodle の指示に従って取り組んでください。

> このリポジトリには受講生由来のデータは含まれていません（個人情報を含むため Moodle 経由で受講者のみに配布）。

### 手順

1. **仮想環境を有効化する**

    ```bash
    conda activate ns_env
    ```

2. **JupyterLab を起動する**

    ```bash
    jupyter lab
    ```

3. **課題に取り組む** — ノートブック `ns1a_XX.ipynb` を開き、指示に従って `.py` ファイルを編集します。ノートブック内で以下のように実行できます。

    ```python
    %run ns1a_XX_01.py
    ```

4. **回答を提出する** — Moodle コースの小テストに回答します。

5. **作業を終えたら無効化する**

    ```bash
    conda deactivate
    ```
