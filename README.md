# ネットワーク科学と経済 I

金沢学院大学 経済情報学部「ネットワーク科学と経済 I」（学部4年次配当・担当: 後藤弘光）の演習課題です。

グラフ理論の基礎からネットワーク分析の実践までを、Python（[NetworkX](https://networkx.org/)）を使って学びます。意図的に誤りを含むスクリプトを修正しながら、ネットワーク科学の概念とプログラミングを同時に身につける構成です。

> 続編: [ネットワーク科学と経済 II](https://github.com/hgphys/ns2)

## 教材の取得

ターミナルまたは Anaconda Prompt で以下を実行してください。

```bash
git clone https://github.com/hgphys/ns1.git
cd ns1
```

---

## 環境構築（初回のみ）

### 1. Anaconda のインストール

以下のサイトを参考にインストールしてください。

- [Python 環境構築ガイド](https://www.python.jp/install/install.html)
- [Anaconda](https://www.anaconda.com/products/distribution)

> ユーザ名が日本語の場合、インストール先のパスに注意してください。

### 2. 仮想環境の作成

```bash
conda env create -f environment.yml
```

### 3. 仮想環境の有効化

```bash
conda activate ns_env
```

プロンプトに `(ns_env)` と表示されれば OK です。

> この環境は「ネットワーク科学と経済 II」でも共通で使用します。

## 課題の取り組み方

各回のフォルダには以下のファイルが含まれます。

```
ns1a_XX/
├── ns1a_XX.ipynb       # 課題本体（Jupyter Notebook）
├── ns1a_XX_01.py       # 問題1用スクリプト（誤りを含む）
├── ns1a_XX_02.py       # 問題2用スクリプト（誤りを含む）
├── ns1a_XX_03.py       # 問題3用スクリプト（誤りを含む）
├── img/                # 画像ファイル（ある場合）
└── edges/              # 辺データファイル（ある場合）
```

### 手順

1. **仮想環境を有効化する**

    ```bash
    conda activate ns_env
    ```

2. **JupyterLab を起動する**

    ```bash
    jupyter lab
    ```

3. **課題に取り組む** — ノートブック `ns1a_XX.ipynb` を開き、指示に従って `.py` ファイルを編集します。ノートブック内で以下のように実行できます。

    ```python
    %run ns1a_XX_01.py
    ```

4. **回答を提出する** — Moodle コースの小テストに回答します。

5. **作業を終えたら無効化する**

    ```bash
    conda deactivate
    ```

## 仮想環境の削除（任意）

すべての授業が終了し環境が不要になった場合は、以下で削除できます。

```bash
conda remove --name ns_env --all
```
